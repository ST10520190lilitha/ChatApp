Project Information  

In MyChatApp part two I developed an application that allows someone to register an account which they have to validate their personal information. The main purpose of the project is to show the importants progamming concepts such as classes and objects, methods, loops, conditional statements, user input, and validation techniques. 

In this appication it has registion and login system. The regristration requires a user to enter their South Afrrica number, name, surname,  password and username. The system validates the information enterd before allowing the user  to continue After successful registration, the user is able to login using their username and password. The program then checks whether the entered details match the stored user information and displays an appropriate login message.

 The application validates the user's credentials by ensuring the username contains an underscore and does not exceed five characters, the password is at least eight characters long and includes an uppercase letter, a number, and a special character, and the phone number begins with the South African international code (+27). If any of these criteria are not met, the system displays an error message and prompts the user to re-enter the information until it is valid. Once successfully registered, the user can log in by providing their credentials, which the system compares against the stored data to either grant access with a welcome message or deny it with an error message for unsuccessful attempts 

 Developed by Lilitha Jubati as part of a student programming assignment, **MyChatApp_part2** is a console-based Java application managed with Maven and built using the NetBeans IDE. The project features a streamlined structure containing `Main.java` to control program execution and `Login.java` to handle user registration, credential validation, and login functionality. Users can easily run the project by opening the folder in NetBeans and pressing F6, allowing them to interact with the system via the console to register with specific credential rules and log in to receive a personalized welcome message. Ultimately, building this application reinforced core programming fundamentals, including object-oriented design, input validation, string manipulation, and runtime error prevention.
